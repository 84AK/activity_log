# 2026년 3월 12일 - 자료실(Resources) 페이지 구현 로그

## 🛠️ 구현된 기능 (Reflector - 작업자, UI Polish - 디자이너, Fix - 해결자)
*   **자료실 메뉴 추가**: 대시보드 화면 상단에 탭 네비게이션을 추가하여 '실습 활동 대시보드'와 '강의 자료실'을 분리했습니다.
*   **권한 분리**: 일반 학생 모드에서는 자료 열람만 가능하며, 관리자(admin) 모드에서는 `자료 등록`, `삭제` 기능이 활성화됩니다.
*   **마크다운 지원**: `react-markdown`과 `remark-gfm`, `@tailwindcss/typography` 플러그인을 도입하여 관리자가 작성한 마크다운 문법(제목, 링크, 강조, 목록 등)을 예쁘게 렌더링합니다.
*   **UI/UX 디자인**: 
    *   자료실 카드는 보라색 계열(`purple`)의 그라데이션과 Border 효과를 주어 기존 실습 기록과 시각적으로 구분되도록 디자인했습니다.
    *   새로운 아이콘(Book, BookOpen) 및 매끄러운 뷰어 전환(Framer Motion) 애니메이션을 적용했습니다.

## 💾 앱스 스크립트 (Apps Script) 데이터베이스
*   기존 실습 활동 시트(`Team, Prompt` 등)와 별개로 `Resources` 전용 시트를 자동 생성하도록 설계했습니다.
*   자료 가져오기(`getResources`), 자료 등록(`createResource`), 자료 삭제(`deleteResource`) API 엔드포인트 핸들러 코드가 포함된 최신 `.gs` 스크립트를 작성하여 제공했습니다.

## 📋 다음 단계 (Next Steps)
1.  관리자(사용자)가 직접 구글 스프레드시트의 Apps Script 편집기에 접속하여 **제공된 전체 `.gs` 코드를 붙여넣고 [새 버전으로 배포]** 해야 합니다.
2.  이후 앱에서 테스트로 실제 관리자 계정으로 로그인 후 자료를 하나 업로드하여 정상적으로 노출되고 삭제되는지 테스트를 진행합니다.
