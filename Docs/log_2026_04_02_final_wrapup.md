# 2026-04-02 최종 작업 내역 통합 로그

## 프로젝트 서기(Scribe) 기록 📝

### 1. 작업 개요 (Objective)
- 실습 활동 기록 카드의 UI 향상과 각종 폼 입력창 편의 기능 고도화
- 관리자 권한(Role)을 Google Spreadsheet와 일원화하여 자동 권한 제어 도입
- 자료실 본문 텍스트 복사 버튼 추가 및 관리자/학생 권한 드롭다운 지원 추가

### 2. 세부 수정/구현 사항 (Implementations & Fixes)

#### 🎨 화면 UI 및 사용자 경험 최적화 (Frontend)
- **카드 디자인 개편**: 딱딱했던 '팀 미정' 블루 박스를 아예 드러내고, 사용자 이름의 첫 글자를 딴 심플하고 깔끔한 이니셜 원형 아이콘으로 대체
- **날짜(Date) 입력 직관성**: 달력 팝업 아이콘을 텍스트 입력창 오른쪽 끝에 명확히 분리해, 사용자가 날짜를 타이핑으로 직접 기입하거나 클릭해서 달력을 띄우는(DatePicker) 두 가지 방식을 모두 지원하도록 수정
- **자료실 본문 [복사] 버튼**: 리스트형 자료실 카드의 중간 부분에 `[복사]` 버튼을 실습 프롬프트 복사 버튼과 통일성 있는 디자인으로 삽입하여, 한 번의 클릭으로 클립보드에 본문을 복사하고 토스트 알림을 통해 피드백을 주도록 구현

#### ⚙️ 권한 에러 해결 및 백엔드(Apps Script) 로직 개선
- **수정 버그 픽스 (ReferenceError)**: `google_apps_script.js` 안에서 실습 기록을 수정할 때 존재하지 않는 `row` 변수를 참조해 뻗던 에러를 타겟팅 후, 정상적으로 선언된 `actualPass` 등을 사용하게 고쳐 안정적 작동 확보
- **Data Validation (드롭다운) 기능**: 
  - 앱스 스크립트로 `Users` 시트를 새로 생성하거나 갱신할 때 `Role` 컬럼 전체에 대해 `['admin', 'student']` 값만 허용되게 하는 리스트 드롭다운 형태의 유효성 검사 규칙 적용
  - 회원가입 할 경우 `Role` 컬럼 쪽에 값이 비어있지 않도록 자동으로 `student`를 기본값으로 박아주게 만들어, 사용자의 수고와 오입력을 원천 봉쇄
- **관리자 전용 보안 로그인 통합**: 
  - 과거 주소창에 직접 `/admin`을 쳐서 들어가던 독립적인 방식과 전용 로그인 입력 폼을 모두 파괴. 세션 값을 보유하지 않은 일반 접근은 자물쇠 화면과 함께 [접근 통제] 화면으로 메인 홈페이지로 내쫓는 보안 적용
  - 오직 메인 페이지에서 아이디/비번을 통해 정상 로그인된 사람이 시스템 상 `admin` 권한을 받았을 경우에만 `sessionStorage`에 비밀번호 및 권한 플래그를 넘겨주고, 이 덕분에 화면 상단 우측에 나타나는 **[관리자 화면 이동]** 전용 포탈 버튼으로만 탑승하게 만듦

### 3. 공유/참고 사항 (Notes for Next Step)
- 추후 백엔드 변경 사항이 발생할 때는 꼭 `Docs/google_apps_script.js` 전체 코드를 복사해서 **구글 스프레드시트 -> 확장 프로그램 -> Apps Script 편집기**에 붙여넣고 [새 배포]를 해야 서버에 즉시 적용되는 점을 잊지 마세요!
