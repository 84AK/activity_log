## 📋 개요
교육부의 '학습지원 소프트웨어 필수기준'을 준수하여 아크랩스 AI 실습 포털에 개인정보 안내 팝업 및 상세 처리방침 노출 기능을 완비했습니다.

---

## 🛠️ 수정 및 구현 내용

### 1. 기획 및 정책
- **`Docs/PrivacyPolicy.md`**: 상세 개인정보 처리방침 초안 마련 (수집 목적: 학습 이력 관리, 수집 항목: ID/PW 최소화, 14세 미만 동의 안내 포함).

### 2. 컴포넌트 개발 (`src/components/`)
- **`PrivacyModal.tsx`**: 
  - 최초 진입 또는 로그인 시 1회 노출되는 요약 안내 팝업.
  - Glassmorphism 2.0 기반의 세련된 다크 모드 카드 스타일.
  - `localStorage`를 이용한 '오늘 하루 보지 않기' 기능 탑재.
- **`PrivacyPolicyModal.tsx`**: `ReactMarkdown`을 활용해 상세 방침 번들을 보여주는 모달.
- **`Footer.tsx`**: 상시 접근 가능한 하단 고정 푸터. '개인정보 처리방침' 연결 단추 제공.

### 3. 페이지 연동 (`src/app/page.tsx`)
- 기존 1000라인의 거대 단일 페이지 모듈에서 개인정보 노출 로직(`isPrivacyModalOpen` 등)을 수입하여 주입했습니다.
